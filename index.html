Unterrichtsmaterialien für GPG

Anleitung zur Verwendung mit Miro

  1. Wähle unten das gewünschte Material
  2. Kopiere die URL des Materials
  3. In Miro: Wähle "Tools, Medien und Integrationen" → "iFrame Code einbetten"
  4. Füge die URL ein und passe die Größe an
